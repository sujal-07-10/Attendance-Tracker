#  Attendance Tracker System

##  Overview
The **Attendance Tracker System** is a simple desktop-based application developed using **Python** and **Tkinter**. It helps in recording and managing student attendance in an organized and efficient way.

This project demonstrates the practical implementation of **Data Structures (Dictionary & List)** along with a **Graphical User Interface (GUI)**.

---

##  Features
-  Mark attendance (Present/Absent) for any date
-  Check attendance of a student for a specific date
-  Date-wise attendance record management
-  Input validation (empty input handling)
-  User-friendly GUI interface

---

##  Technologies Used
- **Programming Language:** Python  
- **GUI Library:** Tkinter  
- **Data Structures:**  
  - Dictionary  
  - List  

---
## Objectives
To develop a simple attendance tracking system

-To apply data structure concepts in real-world scenarios

-To create a user-friendly GUI

-To reduce manual errors in attendance management
